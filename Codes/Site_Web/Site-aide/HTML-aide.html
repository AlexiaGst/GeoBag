Indications d'utilisation

  • GPS:

    • Le signal gps fonctionne en extérieur, les données renvoyées en intérieur ne sont donc pas toujours fiables. Dans ce cas utiliser le mode GATEWAYS.

    La position n'est pas précise, prévoir un rayon d'une vingtaine de mètres autour du point.

    La position mettre un temps avant de s'actualiser.

  • GATEWAYS:

    • Le mode GATWAYS ne fonctionne qu'avec un minimum de 2 gateways captées.

    Si le nombre de gateways est inférieur à 2, la localisation est sur la gateway captée.
